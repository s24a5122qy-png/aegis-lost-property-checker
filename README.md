# A.E.G.I.S. (Automated Equipment & Gear Inspection System)
### 近未来装備スキャナー & 忘れ物チェッカー (Version 1.2.0)

A.E.G.I.S.（イージス）は、サイバーパンクなSF世界観を日常に融合させた、インタラクティブな忘れ物チェックシステム（Web-SPA）です。エージェント（ユーザー）が日々のミッション（通勤、出張、旅行）へ出発する前に、必要な装備（持ち物）を確実に確保するための支援を行います。

---

## 🌌 フレーバー設定 (Backstory)
> "エージェント、今日のミッションの準備は万全か？"
> ゲートが開くその前に、君の装備はすべて揃っているだろうか。
> 降水確率、出発制限時間、持ち物の安全確認（SECURED）。
> すべてのプロトコルがクリアされたとき、イージスは外殻防御シールドを展開し、君の出撃を許可する。

---

## 🛠️ 主要な機能

### 1. インタラクティブ・スキャンコンソール
- **ホログラフィック進捗サークル**: 装備のチェック完了率を円形SVGアニメーションでリアルタイム表示。背後で2つのリングが逆回転し、ホログラムの質感を表現。
- **Web Audioシンセサイザー効果音**: 外部アセット不要で、ブラウザ内で効果音（起動、クリック、チェック、全確保、警告）を合成して再生します。

### 2. サイバー気象センサー (WEATHER MONITOR)
- 天候を「🌧️ RAINY」に切り替えると、AIが警告音とともに「雨具 (RAIN SHIELD)」を自動でリストの最上部へデプロイ（追加）します。晴れや曇りに戻すと自動的に回収されます。

### 3. 出発カウントダウン & アラート (DEPARTURE TIMER)
- 目標の出発予定時刻をセットすると、ミリ秒単位でカウントダウンします。
- 残り時間が1分未満になると自動的にシステムが「RED ALERT（赤い点滅）」を発動し、警告サイレンとAIによる音声警告が流れます。
- 装備がすべて揃うとタイマーは停止し、アラートはクリアされます。

### 4. SF風スキャンカメラHUD (GEAR HUD SCANNER)
- カメラ映像の上に、サイバーな照準HUD、走査レーザー、走査グリッドをオーバーレイ。
- 「RUN SCAN」を押すと、シャッター音とフラッシュ演出が走り、コンソールにダミーの高速文字解析結果を表示。未確保の持ち物が自動的にチェック状態（スキャン完了）に更新されます。

### 5. AIコア切り替え & 音声モデルカスタマイズ
- 3つのAIコア（人格）を選択可能。それぞれ声のトーンや速度、セリフの言い回しが変化し、インターフェースのテーマカラーがキャラクターの色に連動します。
  - **A.E.G.I.S. (Standard)**: 標準AI（女性ボイス / シアンテーマ）
  - **K.R.O.N.O.S. (Military)**: 冷徹な軍用AI（低音の男性ボイス / レッドテーマ）
  - **L.U.N.A. (Support)**: 明るい相棒AI（高めの女性ボイス / マゼンタテーマ）
- **VOICE MODEL**: OS/ブラウザに登録されている日本語音声合成エンジン（Microsoft Haruka, Ichiro, Ayumi など）を自動検出し、手動でボイスを固定・変更できます。

## 💻 技術スタック
- **構造**: HTML5 (セマンティックタグ)
- **デザイン**: Vanilla CSS (CSS Variables, Clip-path, Backdrop-filter, Webkit-scrollbar, Grid layout)
- **ロジック**: Pure JavaScript
- **オーディオ合成**: Web Audio API (OscillatorNode, BiquadFilterNode, GainNode, AudioBuffer)
- **音声合成**: Web Speech API (SpeechSynthesis, SpeechSynthesisUtterance)
- **カメラ処理**: MediaDevices API (getUserMedia WebRTC)
- **ホスティング/自動デプロイ**: Vercel (GitHub連携によるPushトリガーデプロイ)
